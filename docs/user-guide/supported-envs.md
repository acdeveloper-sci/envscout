# Supported Environment Types

EnvScout detects the following Python virtual environment types:

| Type         | Tool                 | Status      |
| ------------ | -------------------- | ----------- |
| `venv`       | Python stdlib `venv` | ✅ Supported |
| `virtualenv` | `virtualenv` package | ✅ Supported |
| `conda`      | Anaconda / Miniconda | ✅ Supported |
| `poetry`     | Poetry               | ✅ Supported |
| `pipenv`     | Pipenv               | ✅ Supported |
| `tox`        | tox                  | ✅ Supported |
| `uv`         | Astral uv            | ✅ Supported |
| `hatch`      | Hatch                | 🔄 Planned   |
| `rye`        | Rye                  | 🔄 Planned   |
